This tool was written in SAS with the help of JCL under mainframe to automate the process. 

This automated tool has following features:
	1. runs every Monday when the user logs into Mainframe terminal
	2. record count check of 4 directories 
	3. compare a directory to its previous to see changes
	4. generate code that will move files in a batch
	5. send email summary to user.
